How to Read This Map

  • Node size reflects connection count — larger nodes are more interconnected
  • Node color indicates category (economic, climate, geopolitical, technological, biological, cross-cutting)
  • Edge thickness indicates connection strength (1-5 scale based on evidence)
  • Arrows show directionality — how risk propagates
  • Red edges are amplifying connections — they make things worse
  • Dashed edges are dampening (blue) or contingent (gray)

What This Map Shows

This map visualizes the interconnections between 20 operational risk themes tracked by The Fragility Brief. Each connection is backed by evidence — historical cases, academic literature, or expert assessment — and weighted on a 1-5 scale for strength.

The map is not a prediction. It shows pathways through which risk can propagate, helping you think in systems rather than silos.
